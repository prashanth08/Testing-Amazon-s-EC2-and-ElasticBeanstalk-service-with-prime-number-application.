Basic server app for testing prime numbers through login functionality. This is a simple webapp implementing login functionality, built on Flask. It implements sign up, login, and a prime number calculation.It is best to be used under virtualenv.


INSTALLATION
------------
Unzip the folder and type in the following command:
sudo apt-get install python-virtualenv
virtualenv flaskapp
If you use virtualenv, go ahead and set that up.  Then use the requirements.txt file to install the dependencies (pip install -r requirements.txt)
cd flaskapp
pip install flask
python application.py -> http://localhost:5000
